# 汽车厂商相关接口

## 主要功能

- 获取汽车厂商列表

# 初始化

+ object  $client  HepcClient对象
+ integer $brandId 汽车品牌Id

```php
$carManu = new CarManu($client, $brandsId);//创建实例对象
```

# 接口

## 获取汽车厂商列表  

### `CarManu::getCarManus()` 

#### 调用示例

```php
$client = new HepcClient($host, $key, $secret, $opts);
$carManu = new CarManu($client, $brandsId);
$ret = $carManu->getCarManus();
```

### 返回示例

```php
Array
(
    [0] => Array
        (
            [id] => 4
            [manu_name] => 一汽大众(奥迪)
        )
    [1] => Array
        ...
)
```
